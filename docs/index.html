Q:如何使用洁面乳?

核心技术


EV+

天然植物萃取成分，对电磁辐射具有高饱和度吸收能力，滋润肌肤的同时，与皮肤表面形成透气保护层，持续有效代替肌肤表皮吸收电磁辐射，有效缓解肌肤所遭受的直接辐射伤害。

D-leon

超临界CO2萃取技术提取高纯度嫩肤成分，通过OCO水体系透过肌肤表层，为肌肤提供充沛营养动力，服感光滑、亮泽、具有显著的立体透白，紧肤功效。

OCO2.0

OCO小分子水体系的基础上，进一步通过调整氢键的矩阵结构，优化磁处理过程，使小分子水半径进一步缩小，排列更密集，从而更容易深入肌肤底层细胞，传递肌肤营养成分。
